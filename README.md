# Projet NLP ENSAE 2024

Ce dépôt contient le code, les données ainsi que les figures associées au projet "Prédiction du sexe à partir des données personnelles" réalisé pour le cours de NLP de l'ENSAE Paris.

Afin d'exécuter le code, voici les commandes à lancer dans un terminal:
````
git clone https://github.com/pbarbarant/nlp-ensae-2024.git
cd nlp-ensae-2024
pip install -e .
````

Pour lancer le preprocessing:
````
python data/preprocessing.py
````

Le notebook `classification.ipynb` contient l'ensemble du code nécessaire pour générer prédictions, évaluations et figures apparaissant dans le rapport.
